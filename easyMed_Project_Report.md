## 1. Project Overview
easyMed is a comprehensive multi-specialty telemedicine platform designed to bridge the gap between rural patients and specialized medical care. The platform facilitates real-time consultations, AI-driven symptom analysis, and secure medical record management.

## 2. Key Features
- **URD Symptom Diagnostic Tool**: Clinical decision support for Urinary, Respiratory, and Diarrheal symptoms.
- **AI Assistant**: Real-time AI chatbot for health inquiries and medical guidance.
- **Case Management**: End-to-end tracking of patient cases from rural doctors to specialists.
- **DICOM/Imaging Viewer**: In-browser medical imaging viewer for diagnostic accuracy.
- **Real-time Chat**: Secure communication channel between patients and doctors.

## 3. Technology Stack
- **Backend**: Python, Flask, Flask-SocketIO
- **Database**: PostgreSQL (Production), SQLite (Local)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Mobile**: Native Android (WebView Integration)
- **Deployment**: Render, Google Play Console

## 4. Implementation Details: URD Symptom Tool
The URD Symptom tool provides a wizard-style diagnostic workflow. It uses strict clinical branching logic to:
- Assess Urinary symptoms (dysuria, frequency, urgency).
- Detect Red Flags (Respiratory rate, O2 saturation, BP, etc.).
- Determine clinical outcomes: Uncomplicated UTI, Complicated UTI, or Medical Emergency.
